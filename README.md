
# 📊 Sales Analysis Dashboard using Microsoft Excel

An interactive Sales Analysis Dashboard built entirely in **Microsoft Excel** using Pivot Tables, Pivot Charts, Slicers, KPI Cards, and Excel formulas.

This project analyzes sales performance, customer spending, product revenue, city-wise orders, and seasonal trends to help businesses make data-driven decisions.

# 📌 Project Overview

The dashboard provides an overview of sales performance through interactive visualizations.

Users can filter data using slicers to analyze sales by:

- Occasion
- Order Date
- Delivery Date

The dashboard automatically updates all charts and KPI cards based on the selected filters.

---

# 📈 Dashboard KPIs

- Total Orders
- Total Revenue
- Average Customer Spending
- Average Order to Delivery Time

---

# 📊 Dashboard Visualizations

### Revenue by Category
Shows revenue generated from each product category.

### Revenue by Occasion
Analyzes sales during different occasions such as:

- Anniversary
- Birthday
- Diwali
- Holi
- Raksha Bandhan
- Valentine's Day

### Revenue by Order Time
Displays hourly order trends to identify peak ordering hours.

### Monthly Revenue
Shows month-wise revenue trends.

### Top 10 Products by Revenue
Ranks products based on generated revenue.

### Top 10 Cities by Orders
Displays cities with the highest number of orders.
